Latihan soal

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun struktur halaman web yang bisa ditampilkan di browser internet. HTML bukanlah bahasa pemrograman seperti JavaScript atau Python, melainkan bahasa markup yang berfungsi untuk menandai dan mengatur bagian-bagian dari sebuah halaman \web, seperti judul, paragraf, gambar, tautan, tabel, dan elemen lainnya.

Fungsi utama HTML adalah memberikan struktur pada sebuah halaman web. Dengan HTML, kita bisa menentukan mana yang merupakan judul halaman, paragraf, gambar, tautan (link), tabel, daftar, dan elemen-elemen lainnya. Setiap bagian dari halaman web ditandai dengan tag HTML tertentu, misalnya:

Berikut contoh elemen HTML yang umum digunakan:

  • <h> : Untuk judul atau heading
  • <p> : Untuk paragraf
  • <a> : Untuk tautan atau link
  • <img> : Untuk gambar
  • <ul>, <ol>, <li> : Untuk daftar

HTML bekerja dengan cara memberikan instruksi kepada browser tentang bagaimana sebuah konten harus ditampilkan. Misalnya, jika kamu ingin menampilkan sebuah paragraf, kamu cukup menuliskan teks di antara tag <p> dan </p>. Browser akan membaca tag tersebut dan menampilkan teks sebagai sebuah paragraf.

Tanpa HTML, sebuah halaman web tidak akan memiliki struktur yang jelas. Semua konten akan terlihat berantakan dan sulit dibaca. HTML juga memudahkan pengembang web untuk menambahkan elemen-elemen lain seperti gambar, video, formulir, dan sebagainya. Selain itu, HTML juga menjadi dasar bagi teknologi web lainnya seperti CSS (untuk mempercantik tampilan) dan JavaScript (untuk membuat halaman menjadi interaktif).

Sejarah HTML

HTML (HyperText Markup Language) pertama kali dikembangkan oleh Tim Berners-Lee, seorang fisikawan di CERN (Organisasi Eropa untuk Riset Nuklir), Swiss. Pada tahun 1989, Berners-Lee mengusulkan sebuah sistem berbasis hypertext untuk memudahkan para ilmuwan CERN dalam berbagi dan mengakses dokumen secara efisien melalui jaringan komputer lalu pada tahun 1991 tim benner-le secara resmi memperkenalkan HTML pada khalayak umum. Sebelum lanjut membahas sejarah makin jauh kita harus mengenal siapa sih itu Tim Berners-Lee, Tim Berners-Lee adalah seorang ilmuwan komputer dan fisikawan asal Inggris yang dikenal luas sebagai pencipta World Wide Web (WWW). Ia lahir pada 8 Juni 1955 di London dan menempuh pendidikan tinggi di Universitas Oxford, di mana ia lulus dengan gelar di bidang fisika pada tahun 1976. Sejak muda, Berners-Lee telah menunjukkan minat dan kreativitas dalam bidang teknologi , bahkan merakit komputer pertamanya dari komponen televisi bekas.

Pada akhir tahun 1991, Tim Berners-Lee merilis dokumen pertama yang menjelaskan tentang HTML, yang dikenal sebagai "HTML Tags". Dokumen ini memuat 18 tag dasar yang menjadi fondasi awal HTML. HTML sendiri awalnya dirancang berdasarkan SGML (Standard Generalized Markup Language), sebuah standar internasional untuk pembuatan dokumen dengan tanda markup.

Perkembangan HTML dari versi awal ke versi sekarang :

  • HTML 1.0 : secara resmi dipublikasikan pada tahun 1993 oleh Internet Engineering Task Force (IETF). Versi ini menjadi standar awal bagi penulisan dokumen web yang dapat dibaca oleh browser. Pada masa itu, browser pertama yang mendukung HTML adalah WorldWideWeb (kemudian berganti nama menjadi Nexus), diikuti oleh browser lain seperti Lynx, Mosaic, dan Arena.
  • HTML 2.0 :Dirilis pada November 1995 oleh IETF sebagai penyempurnaan dari HTML+ (1993). Versi ini menjadi standar resmi pertama HTML yang banyak digunakan.
  • HTML 3.2 : Pada Januari 1997, World Wide Web Consortium (W3C) mengambil alih pengembangan HTML dan merilis HTML 3.2, memperkenalkan banyak fitur baru dan menjadi standar web global.
  • HTML 4.0 dan 4.1 : Dirilis pada Desember 1997 dan April 1998, membawa pembaruan besar seperti pemisahan antara struktur (HTML) dan tampilan (CSS), serta penghapusan beberapa elemen yang sudah usang.
  • HTML 5 : Mulai dikembangkan sejak 2008 dan resmi dirilis pada Oktober 2014, HTML5 membawa banyak fitur baru seperti dukungan multimedia (audio, video), grafis, serta peningkatan integrasi dengan CSS dan JavaScript. HTML5 menjadi standar utama yang digunakan hingga saat ini.

Sejak awal kemunculannya, HTML telah menjadi pondasi utama dalam pengembangan web. Standarisasi dan pengembangannya kini dikelola oleh W3C, memastikan HTML terus relevan dan mampu memenuhi kebutuhan teknologi web modern

Tools untuk menulis HTML

  • Visual Studio Code

    • Visual Studio Code adalah editor kode modern yang sangat populer dan sering digunakan oleh pemula hingga profesional. Aplikasi ini bisa kamu unduh secara gratis dan berjalan di Windows, macOS, maupun Linux. VS Code punya tampilan yang sederhana namun fitur-fiturnya sangat lengkap. Salah satu fitur terbaiknya adalah IntelliSense, yaitu sistem yang bisa menebak dan menyarankan kode yang kamu ketik, sehingga kamu tidak perlu mengingat semua perintah HTML. Selain itu, VS Code juga bisa memperingatkan jika ada kesalahan dalam kode dan bisa langsung digunakan untuk mengecek hasilnya. Kamu juga bisa menambahkan ekstensi untuk bahasa pemrograman atau tema sesuai kebutuhanmu.

    Notes : Cocok untuk Pemula yang ingin belajar HTML dengan bantuan fitur pintar yang memudahkan.

  • Sublime Text

    • Sublime Text adalah editor yang terkenal karena ringan dan sangat cepat. Meskipun tampilannya sederhana, editor ini memiliki banyak fitur keren seperti multiple selection, yang memungkinkan kamu mengedit beberapa bagian kode secara bersamaan. Walaupun Sublime Text bisa digunakan secara gratis, sebenarnya ia adalah software berbayar. Namun, kamu tetap bisa menggunakannya tanpa membayar, hanya akan muncul pengingat sesekali.

    Notes : Cocok untuk Pemula yang ingin editor cepat, ringan, dan tidak terlalu banyak fitur yang membingungkan.

  • Notepad++

    • Notepad++ adalah editor teks ringan dan cepat yang hanya tersedia untuk sistem operasi Windows. Meski sederhana, Notepad++ memiliki fitur penting seperti syntax highlighting, multi-tab editing, dan pencarian cepat yang membuat penulisan kode menjadi lebih mudah. Editor ini juga mendukung plugin untuk menambahkan fungsionalitas tambahan. Notepad++ sangat cocok bagi pemula atau siapa saja yang membutuhkan editor dasar tanpa terlalu banyak fitur tambahan.

    Notes : Cocok untuk Pengguna baru yang ingin mulai belajar HTML dari editor yang simpel.

List dan Tabel

Belajar membuat list terurut dan tidak terurut beserta membuat tabel.

Latihan soal

Ordered dan unordered list

Dalam HTML, daftar atau list digunakan untuk menampilkan sejumlah item secara berurutan atau tidak berurutan. Terdapat dua jenis utama list, yaitu ordered list dan unordered list.

Ordered List (daftar terurut) menggunakan tag <ol> dan biasanya menampilkan item dengan angka atau huruf. Ordered list digunakan saat urutan item penting atau memiliki makna tertentu, seperti langkah-langkah dalam instruksi atau urutan peringkat.

Unordered List (daftar tidak terurut) menggunakan tag <ol>, dan biasanya menampilkan item dengan simbol seperti titik (bullet), lingkaran, atau kotak. Daftar ini cocok digunakan untuk item yang tidak memiliki urutan tertentu.

Struktur Tabel dalam HTML

Tabel dalam HTML digunakan untuk menampilkan data dalam bentuk baris dan kolom. Tag utama yang digunakan dalam pembuatan tabel meliputi <table>, <tr> (table row), <th> (table header), dan <td> (table data).


Penjelasan pada codingan diatas

  • <table> adalah elemen utama untuk membuat tabel.
  • <tr> mewakili satu baris.
  • <th> digunakan untuk membuat sel header yang tampil tebal dan rata tengah secara default.
  • <td> digunakan untuk sel data biasa.

Tabel di atas menampilkan dua data pelajar dengan tiga kolom: Nama, Umur, dan Alamat.

  • rowspan digunakan untuk membuat satu sel mencakup beberapa baris.

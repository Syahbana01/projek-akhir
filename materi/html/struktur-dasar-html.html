Struktur Dasar HTML

Mempelajari struktur dasar dokumen HTML dan tag-tag penting.

Latihan soal

Doctype dan Tag HTML

<Doctype html> adalah deklarasi tipe dokumen (doctype) yang digunakan pada baris pertama dalam dokumen HTML. Deklarasi ini berfungsi untuk memberi tahu browser bahwa halaman web ditulis menggunakan standar HTML5, yaitu versi HTML terbaru dan paling umum digunakan saat ini. Tujuan utama dari <Doctype html> adalah untuk memastikan browser menampilkan halaman web dalam mode standar (standards mode). Dengan begitu, tampilan halaman akan konsisten di berbagai browser dan perangkat. Jika deklarasi ini tidak dituliskan, browser bisa masuk ke quirks mode, yaitu mode tampilan lama yang tidak sepenuhnya mengikuti standar HTML dan CSS. Hal ini dapat menyebabkan halaman web terlihat berbeda atau rusak di beberapa browser.


<Doctype html> adalah deklarasi penting yang digunakan di awal dokumen HTML untuk memberi tahu browser bahwa dokumen tersebut menggunakan standar HTML5. Dengan menyertakan deklarasi ini, browser akan menampilkan halaman web dalam mode standar, yang memastikan tampilan konsisten dan sesuai aturan HTML/CSS modern. Meskipun sederhana <Doctype html> berperan besar dalam menjaga kompatibilitas dan kestabilan tampilan halaman di berbagai browser. Oleh karena itu, deklarasi ini wajib dituliskan pada setiap dokumen HTML.

Tag Head dan Body

Tag Head

Tag <head> adalah bagian dari dokumen HTML yang berisi informasi meta atau informasi pendukung tentang halaman web, bukan konten utama yang ditampilkan di browser. Isi dari tidak langsung terlihat oleh pengguna, tapi sangat penting bagi browser dan mesin pencari.Isi umum yang ada dalam <head≶:

  • <title> -> Menentukan judul halaman yang muncul di tab browser.
  • <meta> -> Memberikan informasi tambahan seperti charset (karakter encoding), deskripsi, dan kata kunci halaman.
  • <link> -> Menghubungkan halaman ke file eksternal seperti CSS (stylesheet)
  • <style> -> Menyisipkan CSS langsung dalam dokumen.
  • <script> -> Menyisipkan atau menghubungkan JavaScript.

Tag <head> dalam HTML berfungsi sebagai tempat untuk menyimpan informasi penting tentang halaman web yang tidak ditampilkan secara langsung kepada pengguna, tetapi dibutuhkan oleh browser dan mesin pencari. Elemen-elemen seperti <title>, <meta>, <link>, dan <sctipt> biasanya ditempatkan di dalamnya untuk mengatur judul halaman, pengkodean karakter, pemanggilan file CSS atau JavaScript, dan informasi pendukung lainnya. Dengan kata lain, tag <head> berperan sebagai bagian "pengatur" dari halaman web, yang membantu memastikan bahwa halaman dapat ditampilkan dan berfungsi dengan baik sesuai standar web.

Tag Body

Tag <body> adalah bagian utama dari dokumen HTML yang berisi semua konten yang akan ditampilkan di layar oleh browser. Apa pun yang ingin kamu tampilkan kepada pengguna (teks, gambar, video, tombol, dll) harus diletakkan di dalam tag <body>.

  • <p> -> Berupa teks paragraf pada bagian body
  • <h1>-<h6> -> Menentukan judul pada bagian isi atau body
  • <img> -> Untuk mengimport gambar
  • <table> -> Menginisiasikan tabel pada HTML
  • <form> -> Untuk menyajikan formulir
  • <button> -> Untuk memunculkan fitur tombol pada body

Tag <body> adalah bagian utama dari dokumen HTML yang berisi seluruh konten yang akan ditampilkan kepada pengguna di browser, seperti teks, gambar, video, tombol, dan elemen-elemen interaktif lainnya. Semua yang ingin dilihat dan digunakan oleh pengunjung website harus diletakkan di dalam tag ini. Dengan kata lain, <body> merupakan "tubuh" dari halaman web yang menyajikan isi sebenarnya kepada pengguna.

Hubungan <head> dan <body>

Tag <head> dan <body> dalam HTML bekerja sama untuk membentuk halaman web yang lengkap. Tag <head> berisi informasi pendukung seperti judul halaman, metadata, dan link ke CSS atau JavaScript yang diperlukan agar halaman berfungsi dengan baik. Sementara itu, tag <body> berisi semua konten yang ditampilkan kepada pengguna, seperti teks, gambar, dan tombol. Keduanya saling melengkapi <head> mengatur di balik layar, sedangkan <body>.> menampilkan hasilnya di layar.

Metadata dan Title

Metadata

Metadata adalah informasi tentang data, atau lebih tepatnya, informasi tambahan tentang halaman web yang digunakan oleh browser, mesin pencari (seperti Google), dan media sosial. Metadata tidak ditampilkan langsung di halaman web, tapi penting untuk fungsi teknis dan SEO (Search Engine Optimization). Metadata ditempatkan di dalam tag <head> dan biasanya ditulis menggunakan tag <meta>.


Metadata ditulis menggunakan tag <meta> dan diletakkan di dalam bagian <head> pada dokumen HTML. Beberapa contoh metadata termasuk pengaturan jenis huruf (charset), deskripsi halaman, nama penulis, kata kunci, dan pengaturan tampilan di perangkat mobile.

Title

Tag <title> adalah elemen HTML yang digunakan untuk memberikan judul pada halaman web. Tag ini tidak menampilkan teksnya langsung di dalam halaman yang dilihat pengguna, tetapi judul ini akan muncul di bagian atas tab browser dan juga akan terlihat sebagai judul utama saat halaman muncul di hasil pencarian mesin pencari seperti Google, Bing, atau Yahoo.


Tag <title> dalam HTML berfungsi untuk memberikan judul halaman web yang tampil di tab browser seperti contoh diatas pada bagian tab browser akan bernama "belajar HTML dasar". Kemudian digunakan sebagai judul utama di hasil pencarian mesin pencari seperti Google. Judul ini membantu pengguna mengenali dan membedakan halaman saat membuka banyak tab, menyimpan bookmark, atau melihat riwayat penelusuran. Selain itu, tag <title> sangat penting untuk SEO dan pengalaman pengguna, karena judul yang jelas dan relevan akan memudahkan orang memahami isi halaman sebelum membukanya. Setiap halaman HTML wajib memiliki satu tag <title>, dan judul yang digunakan sebaiknya singkat, padat, dan sesuai dengan isi konten halaman. Tag ini menjadi identitas penting dari sebuah halaman web, meskipun tidak terlihat langsung dalam isi halaman.

Latihan soal

Tag Form <form> dan Atributnya

Tag <form> digunakan untuk membuat sebuah formulir dalam halaman web, Formulir ini memungkinkan pengguna mengisi dan mengirimkan data.

Berikut Beberapa atribut penting pada tag <form>:

action URL tujuan tempat data formulir dikirim. <form action="/submit"> method Metode pengiriman data, bisa GET atau POST. <form method="POST">

Contoh penggunaan tag <form>:


Tag <a> menggunakan atribut href (hypertext reference) untuk menentukan alamat atau link tujuan yang ingin diarahkan saat tautan diklik

Input Types dan Atributnya

Tag <input> digunakan untuk menerima data dari pengguna. Tipe input menentukan jenis data yang dimasukkan.

Tipe Input Fungsi / Kegunaan Contoh Tampilan
text Input teks biasa
email Input untuk alamat email
password Input untuk sandi (disembunyikan)
radio Pilihan tunggal dari beberapa opsi Pria
checkbox Pilihan ganda Saya setuju
submit Tombol kirim formulir

Label, Button, dan Select

Tag HTML Fungsi / Kegunaan Contoh Tampilan
<label> Memberi keterangan pada input. Gunakan atribut for untuk mengaitkan dengan id input.
<button> Membuat tombol untuk kirim data atau aksi lainnya.
<select> Membuat menu dropdown. Pilihannya dibuat dengan tag <option>.

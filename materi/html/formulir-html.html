Formulir HTML

Mempelajari cara membuat formulir interaktif untuk mengumpulkan data.

Latihan soal

Tag Form <form> dan Atributnya

Saat kamu membuka website toko online, lalu ingin

    1. Mengisi nama dan alamat pengiriman
    2. Memilih metode pembayaran
    3. Menekan tombol "Kirim"
Nah, semua itu kamu lakukan di dalam formulir, dan di HTML kita menggunakan tag <form> untuk membuatnya. Form itu seperti wadah untuk elemen inputan yang akan dikirimkan ke server.

Tag <form> memiliki beberapa atribut penting:

  • action: Alamat URL tujuan saat formulir dikirimkan.
  • method: Metode pengiriman data, biasanya GET atau POST.
  • enctype: Tipe pengkodean data yang digunakan saat mengirimkan formulir (misalnya, untuk file upload).

    • Input Types dan Atribut

    Tag <input> digunakan untuk membuat kolom isi dalam form. Tapi kamu tahu ga? Satu tag <input> bisa jadi macam-macam bentuk hanya dengan mengganti type-nya!

    Tipe Input Fungsinya
    text Input teks biasa(nama, alamat, dll).
    email Input untuk alamat email, ada validasi otomatis (harus pakai @).
    password Input untuk sandi (karakter disembunyikan pakai bintang).
    number Input untuk angka, bisa diatur batasnya (min, max).
    date, time Input untuk memilih tanggal (ada kalender). Dan jam.
    radio Pilihan tunggal (hanya satu yang bisa dipilih)
    checkbox Pilihan ganda (boleh lebih dari satu)
    submit Tombol untuk mengirim formulir
    reset Tombol untuk mengatur ulang semua input ke nilai awal.
    file Input untuk mengunggah file (gambar, dokumen, dll).

    Selain tipe input, ada juga atribut penting yang bisa kamu tambahkan ke tag <input> untuk mengatur perilakunya:

    Atribut Fungsinya
    <label> Teks contoh di dalam kotak isian (seperti "Masukkan nama lengkap").
    <required> Wajib diisi. Kalau kosong, form tidak bisa dikirim.
    <value> Nilai awal (bisa untuk input tersembunyi atau pilihan default).
    <checked> Menandai checkbox atau radio sudah diisi.
    <min / max> Batas angka atau tanggal (misal umur antara 1-80) sudah diisi.
    <name> Nama variabel input yang akan dikirim ke server (penting!).

    Label, Button, dan Select

    Label adalah teks yang memberi tahu pengguna apa yang harus diisi. Tapi lebih dari itu, kalau di klik, label bisa langsung mengaktifkan input-nya.

    Button Tombol yang memberi "aksi" pada formulir, untuk mengirim, menghapus atau sekadar menampilkan pesan.

    Select adalah menu dropdown untuk memilih satu dari beberapa pilihan. Pilihan dibuat dengan tag <option>.

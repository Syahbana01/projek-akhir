Link & Gambar

Memahami dasar-dasar HTML dan perannya dalam pembuatan Website.

Latihan soal

Tag anchor <a>

Tag <a> (anchor) digunakan untuk membuat tautan atau link ke halaman lain, bagian tertentu dalam halaman, atau dokumen. Tautan ini bisa berupa teks atau elemen seperti gambar

Berikut contoh penggunaan tag anchor:


Tag <a> menggunakan atribut href (hypertext reference) untuk menentukan alamat atau link tujuan yang ingin diarahkan saat tautan diklik

Tag Gambar <img>

Tag <img> digunakan untuk menampilkan gambar pada halaman web. Berbeda dengan tag <a> yang memiliki tag pembuka dan penutup (<>), tag <img> adalah tag kosong (self-closing).

Berikut contoh penggunaan tag <img>:/p>

Tag <img> membutuhkan atribut src untuk menentukan sumber gambar, dan atribut alt sebagai teks alternatif jika gambar tidak dapat ditampilkan.

Atribut href, src, dan alt

  • href: digunakan di tag <a> untuk menentukan alamat tujuan dari link.
  • src: digunakan di tag <img> untuk menentukan lokasi atau sumber file gambar.
  • alt: digunakan di tag <img> untuk memberikan teks alternatif yang akan ditampilkan jika gambar gagal dimuat.

Link & Gambar

Mempelajari cara menambahkan link dan gambar ke dalam halaman web.

Latihan soal

Tag anchor <a>

Dalam dunia nyata, kita bisa melihat papan petunjuk arah atau tombol "klik disini" yang membawa kita ke tempat baru. Nah, dalam HTML, elemen yang berperan sebagai "pengarah jalan" ini adalah <a> yang disebut juga (anchor tag).

    Tag <a> digunakan untun membuat hyperlink, yaitu tautan yang bisaa mengarahkan pengguna ke:

  • Halaman lain di website yang sama
  • Website lain (eksternal)
  • File seperti PDF, gambar, atau dokumen.
  • Bagian tertentu di dalam halaman yang sama.
  • Email atau nomor telepon.

Berikut contoh penggunaan tag anchor:


Tag <a> itu seperti pintu masuk ke dunia lain. Tanpa tag ini, pengguna hanya akan "teperangkap" di satu halaman web tanpa bisa menjelajah lebih jauh. Dengan tag <a>, kita bisa membuat tautan yang menghubungkan berbagai halaman web, sehingga pengguna dapat dengan mudah berpindah dari satu web ke web lain.

Tag Gambar <img>

Manusia adalah makhluk visual. Gambar bisa menjelaskan banyak hal lebih cepat dari teks.Di HTML, elemen yang digunakan untuk menampilkan gambar adalah tag <img> fungsinya untuk menampilkan gambar pada halaman web. Berbeda dengan tag <a> yang memiliki tag pembuka dan penutup (<>), tag <img> adalah tag kosong (self-closing).

Berikut contoh penggunaan tag <img> :

Tag <img> membutuhkan atribut src untuk menentukan sumber gambar, berfungsi untuk memnentukan sumber gambar, yaitu lokasi file gambar yang ingin ditampilkan. Lokasi ini bisa berupa file lokal dari folder dalam proyek atau juga bisa berupa URL dari gambar yang berada di internet dan atribut alt sebagai teks alternatif jika gambar tidak dapat ditampilkan. Atribut ini berfungsi sebagai pengganti gambar jika gambar gagal dimuat karena alasan tertentu seperti kesalahan penulisan path atau koneksi yang lambat.

Atribut href, src, dan alt

  • href: digunakan di tag <a> untuk menentukan alamat tujuan dari link. Dengan href, kita bisa mengarahkan pengguna ke halaman lain, situs eksternal, atau bagian tertentu dalam halaman yang sama. Tanpa href, link tidak bisa berfungsi. Ini membuat href menjadi kunci penting dalam navigasi web.
  • src: digunakan di tag <img> untuk menentukan sumber gambar yang ingin ditampilkan. Bisa berupa file lokal dalam proyek atau URL gambar dari internet. Jika src salah atau tidak ada, gambar tidak akan muncul di halaman.
  • alt: digunakan di tag <img> untuk memberikan teks alternatif, teks ini muncul saat gambar gagal dimuat dan sangat membantu pengguna yang menggunakan pembaca layar, sekaligus membantu mesin pencari memamahami isi gambar. Dengan alt, website jadi lebih ramah dan inklusif.

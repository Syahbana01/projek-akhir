Selector CSS

Belajar cara memilih elemen HTML yang akan diberi style.

Latihan soal

Dalam CSS, selector adalah alat untuk memilih elemen HTML yang ingin kita beri style khusus. Selector memungkinkan kita untuk menentukan elemen mana yang akan dipengaruhi oleh aturan CSS yang kita buat. Ada beberapa jenis selector yang umum digunakan, element selector, class selector, ID selector, dan kombinasi selector.

Element Selector

Element Selector adalah selector yang digunakan untuk memilih elemen HTML berdasarkan nama tag-nya. Misalnya, jika kita ingin mengubah warna teks dari semua elemen <p> menjadi biru, kita dapat menggunakan selector ini.


Kode diatas membuat contoh semua elemen <p> akan otomatis berwarna biru

Class dan ID selector

Class Selector digunakan untuk memilih satu atau beberapa elemen HTML yang memiliki atribut class. Dalam CSS, kita menuliskannya dengan tanda titik (.) diikuti nama class. Class ini sangat fleksibel karena bisa digunakan lebih dari satu dari berbagai elemen yang berbeda. Ini cocok untuk membuat style yang dapat dugunakan berulang, seperti design layout kolom, tombol dan kartu.


ID Selector digunakan untuk memilih elemen HTML yang memiliki atribut id. Dalam CSS, kita menuliskannya dengan tanda pagar (#) diikuti nama ID. ID ini unik, artinya hanya boleh digunakan pada satu elemen HTML dalam halaman web. ID biasanya digunakan untuk elemen yang spesifik, seperti header atau footer.

Dengan class selector, kita bisa membuat style yang bisa dipakai berulang. Dengan ID selector, kita memberi gaya pada elemen unik.

Kombinasi Selector

Kombinasi Selector digunakan ketika kita ingin menargetkan elemen secara lebih spesifik berdasarkan struktur HMTL-nya atau lebih dari satu kondisi./p>

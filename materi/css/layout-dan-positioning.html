Layout dan Positioning

Mempelajari cara mengatur posisi elemen dalam halaman web.

Latihan soal

Display Property

Properti disply dalam CSS gunanya untuk mengontrol bagaimana elemen ditampilkan di halaman web entah sebagai blok, inline, fleksibel atau bahkan disembunyikan.

  • block → Elemen ditampilkan sebagai blok penuh. Biasanya membuat elemen tampil di baris baru. Contoh: <div>, <p>
  • inline → Elemen tidak memulai baris baru, hanya selebar isi kontennya. Contoh: <span>, <a>
  • inline-block → Gabungan antara inline dan block. Bisa diberi ukuran, tapi tetap sebaris.
  • none → Elemen tidak akan ditampilkan sama sekali.

    • Bayangkan saja ketika kamu punya mainan blok dan sticker. Elemen block itu seperti balok besar yang butuh satu garis penuh. Sedangkan inline seperti sticker kecil yang menempel di baris yang sama dengan lainnya.

    Position (relative, absolute, fixed)

    Apa itu position? Properti position di CSS untuk menentukan cara suatu elemen ditempatkan dalam halaman web. Apakah elemen itu mengikuti alur normal halaman? Atau harus menempel dipojok? Atau bahkan tetap di tempat saat kamu scroll?

    Berikut jenis-jenis position:

    • relative → Elemen tetap di tempatnya, tapi bisa digeser sedikit.
    • absolute → Elemen "menempel" pada posisi tertentu dalam container terdekat.
    • fixed → Elemen tetap di tempatnya, bahkan saat halaman di-scroll.


      • Tips Praktis:

      • Gunakan relative saat kamu ingin sedikit menggeser elemen tanpa mempengaruhi posisi elemen lain.
      • Gunakan absolute untuk membuat elemen "menempel" di pojok container (wadahnya).
      • Gunakan fixed untuk elemen seperti tombol WhatsApp mengambang atau navbar yang selalu terlihat saat di scroll.

      Float dan Clear

      Apa itu Float?

      float digunakan untuk membuat elemen mengapung ke kanan atau ke kiri, sehingga teks atau elemen lain bisa mengalir di sekitarnya. Contohnya seperti saat gambar majalah dikelilingi oleh teks. Tapi hati-hati! kalau dipakai tanpa menggunakan clear, layout bisa berantakan.

    • left Posisi kiri
    • right Posisi kanan
    • none Tidak di posisi atau tidak tergeser

    Apa itu Clear?

    clear adalah pasangan dari float. Properti ini untuk menghindari efek melayang dari elemen sebelumnya, dan memastikan elemen setelahnya tetap normal


    Bayangkan kamu punya gambar yang mengapung di kiri, lalu teks di scroll di sampingnya. Tapi kalau ada elemen lain yang ingin kamu letakkan di bawah gambar itu, kamu harus pakai clear supaya elemen itu tidak melayang.

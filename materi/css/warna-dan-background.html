Warna Background

Mempelajari cara mengatur warna teks dan background.

Latihan soal

Warna dan latar belakang merupakan elemen penting dalam desain web karena berperan besar dalam menciptakan estetika, memperjelas hierarki informasi, dan meningkatkan kenyamanan pengguna saat membaca. CSS menyediakan beragam properti yang bisa digunakan untuk mengatur warna teks, warna latar belakang, gambar latar, dan lainnya dengan sangat fleksibel.

Color Property

Properti color digunakan untuk mengatur warna teks pada elemen HTML. Ini adalah cara paling dasar dalam memperindah teks. Warna dapat diterapkan pada berbagai elemen seperti paragraf, heading, link, dan sebagainya.


Kode di atas akan membuat semua teks dalam elemen <p> berwarna merah.

Namun, kamu tidak hanya terbatas pada nama warna seperti red, blue, atau green. CSS mendukung format warna lain seperti kode heksadesimal (HEX), RGB, dan HSL. Hal ini memberikan kontrol yang lebih presisi terhadap tampilan warna yang diinginkan.

Background Properties

Properti background adalah sekumpulan properti CSS yang digunakan untuk mengatur latar belakang suatu elemen. Ini mencakup warna latar belakang, gambar latar, pengulangan gambar, posisi, dan lainnya. Beberapa properti latar belakang yang umum digunakan adalah:

  • background-color: untuk menentukan warna latar belakang.
  • background-image: untuk menetapkan gambar sebagai latar belakang.
  • background-repeat: untuk mengatur apakah gambar latar diulang atau tidak.
  • background-position: untuk mengatur posisi gambar latar.
  • background-size: untuk mengatur ukuran gambar latar.
  • background: shorthand untuk semua properti di atas.

Dengan kode di atas, latar belakang halaman web akan memiliki warna hijau, dan elemen dengan class .box akan menampilkan gambar yang memiliki latar gambar css dan atribute lainnya.

Nilai warna (HEX, RGB, HSL)

CSS mendukung beberapa format dalam menetapkan warna:

  • HEX menggunakan angka dan huruf untuk mewakili warna dalam format #RRGGBB. Setiap pasangan mewakili nilai merah, hijau, dan biru.
  • RGB (Red, Green, Blue) Format ini menuliskan nilai merah, hijau, dan biru secara eksplisit, dari 0 hingga 255.
  • HSL (Hue, Saturation, Lightness) Format HSL menggunakan konsep warna berdasarkan roda warna. Hue dalam derajat (0–360), Saturation dan Lightness dalam persen (%).
